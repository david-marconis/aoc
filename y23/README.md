# Advent of code solutions 2023

## Input files
Input files must be placed in a directory named `res` next to the `src` directory and have the format `day[x].txt`. Like this:
```text
├── res
│   ├── day1.txt
│   ...
│   └── day25.txt
└── src
    ├── day1.jl
    ...
    └── day25.jl
```

## How to run
```sh
julia src/day1.jl
```
