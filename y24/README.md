# Advent of code solutions 2024

## Input files
Input files must be placed in a directory named `res` next to the `src` directory and have the format `day[x].txt`. Like this:
```text
├── res
│   ├── day1.txt
│   ...
│   └── day25.txt
└── src
    ├── 1/main.go
    ...
    └── 25/main.go
```

## How to run
```sh
go run src/1/main.go
```
